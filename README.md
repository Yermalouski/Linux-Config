## Deployment

To apply configuration:

```bash
  $ cd ~/Downloads
  $ git clone https://github.com/Yermalouski/Linux-Config
  $ cd Linux-Config
  $ mv config fonts ~/
  $ cd ~/
  $ mv config .config
  $ mv fonts .fonts
```
